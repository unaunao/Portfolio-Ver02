    성장하는 웹퍼블리셔

    안녕하세요. 웹퍼블리셔 오운아입니다.
    잘만든 UI는 사용자에게 좋은 웹 경험을 줍니다.
    사용자 친화적인 웹을 만드는 것이 저의 목표입니다.

    웹 개발은 여러 사람이 함께 만들어 가기 때문에 협업과 소통이 중요하다고 생각합니다.
    잘 듣고 열린자세로 일하며, 새로운 기술도 함께 배워 나아가 유연하게 업무를 수행할 수 있는 사람이 되겠습니다.

    • Develop Tech

      • 디자인된 UI를 웹 환경에 맞게 HTML, CSS로 화면에 흡사하게 구현 가능합니다.
      • SEO 검색앤직 최적화 및 접근성을 좋게 하기 위해서 시멘틱 마크업을 합니다.
      • 크로스 브라우징이가능하며 반응형 홈페이지 및 웹 앱 등의 경험이 있습니다.
      • 유지, 보수에 용이 할 수 있도록 중복된 코드를 줄이고 재사용가능한 구조를 지향합니다.
      • 웹 환경의 다양한 효과를 javascript, jQuery 라이브러리를 사용하여 구현 가능합니다.
      • 부트스트랩, Cafe24, imweb을 이용한 웹 서비스 구현도 가능합니다.

    • Design Tech

      • 서비스 방향과 기획의도에 맞게 프로토타입을 만들고 UI 만듭니다.
      • 익숙한 UX에 기반으로 더 편리한 UI를 만들기 위해 고려합니다.
      • 스타일 시스템을 정리하여 디자인 보수 수정에 일관성 있고 빠른 대처를합니다.
      • Photoshop, Illustrator을 이용한 UI에 넣을 이미지 편집, 아이콘 등 다양한 디자인 작업이 가능합니다.
      • XD, Figma를 사용하여 style system, 와이어 프레임, 프로토타이핑을 만듭니다.
      • Canva툴을 이용하여 다양한 웹 콘텐츠를 만들어냅니다.
      • Premiere Pro 를 이용하여 영상편집이 가능합니다.
